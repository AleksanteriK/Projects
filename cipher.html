Where do we need crypting?

As long as we have had any kind of writing system, we also must have had a way to send a message to someone which can only the receiver understand. Due to this we have had to create some kind of a secret "language" and a certain algorithm to decrypt the crypted message.

What is cryptography?

Cryptography is a study of secure communication systems. It is generally about constructing and analyzing protocols that prevent third parties from reading private messages. Modern cryptography is very much based on mathematical theory.
